## [DeepSeek-V4 预览版发布，超越开源模型并增强 Agent 能力](https://t.me/zaihuapd/41185) ⭐️ 9.0/10

DeepSeek-V4 的预览版已正式上线并同步开源，其 Pro 版本在数学、STEM 和竞赛型代码等关键基准测试中超越所有已公开评测的开源模型。同时，Flash 版本提供更经济高效的 API 服务，两个版本都相比前代模型显著增强了 Agent 能力。 此次发布代表了开源大语言模型（LLM）领域的重要研究突破和技术进展，推动了开源模型能力的边界。其增强的 Agent 能力和经济高效的 Flash 版本具有广泛的应用潜力，使先进人工智能在各种任务中更具可访问性和多功能性。 DeepSeek-V4-Pro 模型在性能上追及顶级闭源模型，尤其在复杂问题解决方面表现出色。DeepSeek-V4-Flash 模型虽然提供接近高级模型的推理和 Agent 能力，但通过更小的模型参数和激活，实现了成本效益和速度，使其适用于日常高频服务。

telegram · zaihuapd · May 3, 02:21

**背景**: 大语言模型 Agent 是一种基于大语言模型构建的智能 AI 系统，旨在理解复杂指令并通过与环境交互执行多步骤任务，通常由感知模块、大脑模块和行动模块组成。神经网络激活函数决定了神经网络中节点的输出，其中非线性函数对于高效解决复杂问题至关重要。DeepSeek-V4-Flash 中较小的模型参数和激活有助于降低计算成本并加快推理速度。
